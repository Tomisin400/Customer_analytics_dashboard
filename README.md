## Project Overview
This project presents a comprehensive **Customer Analytics Dashboard** built in Microsoft Excel. The goal is to analyze customer behavior, identify churn risks, understand revenue performance, and uncover actionable insights across 10 global markets (Brazil, USA, India, Germany, Australia, Canada, China, France, Japan, and UK). The dashboard is designed to help businesses make data-driven decisions around customer retention, loyalty, and engagement strategies.



## Problem Statement

Over 60% of customers are either at medium or high churn risk,
yet the business has no clear visibility into who is leaving, why they are leaving,
and which markets, age groups or categories are most affected.

Revenue is being generated â€” but it is not being protected.

This dashboard was built to answer the questions the business has been avoiding:
Who are we losing? Where are we losing them? And what do we do about it ?

## Data Cleaning & Preparation
All data cleaning was performed **manually in Microsoft Excel** without the use of Power Query.

### Steps Taken:

**1. Initial Assessment**
-  **5,000 data rows** were available for analysis.
  
**2. Missing Values**
Missing values were identified across several key columns:
- **Average Order Value** â€” many records had missing values
- **Total Orders** â€” some records had missing values
-  **Last Purchase** â€” some records had missing values
  
Notably, some records had **Total Orders but no Average Order Value**, and some had  **Average Order Value but no Last purchase** â€” both logically inconsistent and indicate data quality issues.

**3. Validation & Flagging**

A **helper validation column** named *"Column Issue Flagged"* was created to identify and tag all inconsistent and blank records. Rather than deleting these rows, they were **flagged and filtered out** during all Pivot Table calculations, KPI computations, and chart creation to ensure analytical accuracy.

**4. Helper Columns & Formulas**

**Revenue**
```Excel
=[@[total_orders]]*[@[avg_order_value]]
```
**Email Engagement**
```Excel
=IF(L2<=20,"Low",IF(L2<=50,"Medium","High"))
```

**Customer Tenure**
```Excel
=IF(YEAR([@[customer_since]])>=2023, "New",IF(YEAR([@[customer_since]])>=2021, "Medium", "Long term"))
```

**Loyalty Level**
```Excel
=IF(P2<=33,"Low",IF(P2<=66,"Medium","High"))
```

**Churn Risk Categories**
```Excel
=IF(R2<=0.2,"Low",IF(R2<=0.5,"Medium","High"))
```
